name: fix-pr-comments
description: Workflow to follow when asked to fix pr comments
---

Fix the linked comments comments, if no comments are specified ASK if all
comments on pr should be fixed.

Workflow:
1. Fix one by one
2. Absorb with jj between each fix
3. Push after all is fixed
4. React with thumpsup if it is fully fixed and I did not make the comment
5. Resolve the thread after pushing
6. Do not comment on the thread
7. Flag anything that must be looked into
